# Tailspin - pixi


A recreation of *Tailspin* (originally created in Flash) using PixiJS.

![Pixi JS logo](/public/logo/pixi.svg)

Pixi v.7, GSAP v.3 & vanilla TypeScript

## Get started

```
$ bun install
$ bun run dev
```

## Build

```
$ bun run build
```

## Preview the build

```
$ bun run preview
```
